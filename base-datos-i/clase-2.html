Bases de Datos I

Clase 2 - Arquitectura de una BD

Arquitectura es la estructura o forma en que algo está ordenado, dispuesto o construido. En el caso de los sistemas de bases de datos está muy influenciada por el sistema informático subyacente en el que se ejecuta. Estos han evolucionado desde un enfoque monolítico a uno modular o compuesto, pueden ser centralizados, cliente/servidor o distribuidos entre otras. Los sistemas de bases de datos también pueden diseñarse para explotar arquitecturas informáticas paralelas.
